# Retracement Bot
![bot](https://github.com/EsauM10/iqoptionbot/assets/33939999/3a928917-b38b-4731-83b5-67b2e31fc6a7)

## Install
```
pip install -U git+https://github.com/EsauM10/iqoptionbot.git
```
## Running the application
```
iqoptionbot run
```
## Building the executable
```
iqoptionbot build
```
